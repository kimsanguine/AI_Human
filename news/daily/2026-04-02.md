## TODAY'S TOP AI NEWS

### 1. Nature, 시각 피질 CNN 5,000배 압축 성공 — "작을수록 설명 가능하다"
**출처:** [Nature — 2026-04-01](https://www.nature.com/articles/s41586-026-10150-1)

마카크 원숭이 시각 피질(V4 영역)의 신경 반응을 예측하는 DNN을 적응형 폐루프(closed-loop) 실험으로 반복 훈련한 뒤, 6,000만 파라미터 모델을 모델 압축(model compression)을 통해 약 12,000 파라미터로 줄였다. 5,000배 경량화에도 예측 정확도는 유사하게 유지됐다. 연구팀은 "거대 모델이 아니어도 뇌를 충분히 설명할 수 있다"며 해석 가능한 소형 신경망 모델의 가능성을 제시했다.

> **학습 연결:** Ch03에서 배우는 합성곱 연산(convolution)은 뇌의 시각 피질이 이미지를 처리하는 방식에서 영감을 받았는데, 이 연구는 그 역방향—즉 CNN으로 실제 시각 피질 뉴런 반응을 역설계하는 과정—을 보여준다. 로컬 수용야(local receptive field)와 특성 맵(feature map) 개념이 뇌 신경망 구조와 얼마나 깊이 대응하는지 직접 확인할 수 있는 사례다.

### 2. Anthropic, npm 배포 실수로 Claude Code CLI 소스 코드 전체 유출 — 50만 줄·미공개 기능 포함
**출처:** [VentureBeat — 2026-03-31](https://venturebeat.com/technology/claude-codes-source-code-appears-to-have-leaked-heres-what-we-know)

Anthropic이 Claude Code v2.1.88을 npm에 배포하면서 내부 디버깅용 소스맵 파일(59.8MB)을 실수로 포함시켰다. 약 2,000개 파일·50만 줄의 전체 소스 코드가 GitHub에 빠르게 미러링돼 수천 스타를 받으며 분석됐다. 유출 코드에는 미공개 기능 'KAIROS'(백그라운드 상시 실행 에이전트, 오류 자동 수정, 푸시 알림), 멀티 에이전트 오케스트레이션, LLM 툴-콜링 아키텍처 전체가 담겼다. Anthropic은 "보안 침해가 아닌 패키징 실수"라고 공식 인정했다.

> **학습 연결:** 유출된 아키텍처의 핵심은 LLM API 호출을 처리하는 쿼리 엔진과, 신경망 출력을 bash·파일 읽기 등 실제 도구 실행으로 연결하는 툴 시스템이다. Ch03에서 배우는 신경망(neural network)이 단순 추론 모델을 넘어 에이전트 오케스트레이션 레이어와 결합될 때 어떻게 실제 개발 도구가 되는지, 그 구조를 소스 코드 수준에서 직접 확인할 수 있는 드문 기회다.

### 3. CNN + 어텐션 메커니즘, 영상 폭력 탐지를 "설명 가능"하게 만들다
**출처:** [Nature Scientific Reports — 2026-04-02](https://www.nature.com/articles/s41598-026-40977-7)

비지도 키프레임 선택(unsupervised keyframe selection)과 어텐션 기반 CNN을 결합한 영상 폭력 탐지 프레임워크가 공개됐다. 어텐션 레이어가 폭력 장면에서 중요한 시각적 특징을 자동 강조하고, 비지도 학습으로 선별한 핵심 프레임만 분석해 연산 효율을 높였다. 기존 블랙박스 모델 대비 설명 가능성(explainability)이 크게 향상됐다.

> **학습 연결:** Ch03에서 배우는 CNN의 합성곱 연산은 이미지·영상의 지역 패턴(local pattern)을 추출하는 데 특화돼 있다. 이 연구는 여기에 어텐션 메커니즘을 더해 "어디에 집중하는가"를 시각화함으로써, 합성곱 필터가 실제로 어떤 공간적 정보를 포착하는지 해석하는 방법을 직접 보여준다.

### 4. Runway, $1,000만 달러 'Builders 펀드' 출범 — 초기 AI 스타트업에 무료 API 크레딧 지원
**출처:** [TechCrunch — 2026-03-31](https://techcrunch.com/2026/03/31/exclusive-runway-launches-10m-fund-builders-program-to-support-early-stage-ai-startups/)

AI 비디오·월드 모델 회사 Runway가 $1,000만 달러 규모의 벤처 펀드와 'Builders 프로그램'을 동시에 출범했다. 시드~시리즈C 스타트업에 50만 API 크레딧을 무료로 제공하며, 투자 철학은 ① 새로운 아키텍처를 연구하는 기술팀 ② 파운데이션 모델 위 애플리케이션 레이어 빌더 ③ 새로운 미디어 형식 실험자 세 범주로 구성된다. 첫 코호트에는 LanceDB, Tamarind Bio 등이 포함됐다.

> **학습 연결:** Runway가 우선 투자하는 첫 번째 범주가 "새로운 딥러닝 아키텍처를 연구하는 팀"이라는 점은, Ch03에서 배우는 신경망 설계 역량이 실제 스타트업 투자 가치로 직결됨을 보여준다. 딥러닝 아키텍처 이해는 AI 제품 개발의 기초 경쟁력이다.

### 5. Bloomberg Intelligence: AI 가속기 시장, 2033년 $6,000억 돌파 — ASIC·맞춤형 칩이 NVIDIA 독점 흔든다
**출처:** [Bloomberg — 2026년 4월 초](https://www.bloomberg.com/company/press/ai-accelerator-market-looks-set-to-exceed-600-billion-by-2033-driven-by-hyperscale-spending-and-asic-adoption-according-to-bloomberg-intelligence/)

Bloomberg Intelligence 보고서에 따르면 AI 가속기 시장이 2033년까지 $6,000억을 초과할 전망이다. 하이퍼스케일러(구글, 메타, 아마존)의 자체 ASIC 설계 및 Broadcom의 맞춤형 칩 채택이 가속화되면서 NVIDIA의 GPU 독점 구도가 흔들리고 있다. Broadcom은 특화 태스크(챗봇 추론 등)에서 NVIDIA 대비 훨씬 저렴한 비용으로 동등한 성능을 제공한다고 밝혔다.

> **학습 연결:** AI 가속기는 PyTorch, TensorFlow 같은 딥러닝 프레임워크가 돌아가는 하드웨어 기반이다. Ch03에서 배우는 CNN·DNN 모델의 행렬 연산(matrix multiplication)은 GPU와 ASIC의 병렬 처리 구조에 최적화돼 있으며, 어떤 칩을 선택하느냐에 따라 학습 속도와 비용이 크게 달라진다는 점을 이 시장 동향이 실감하게 해준다.
