## TODAY'S TOP AI NEWS

### 1. OpenBMB, 토크나이저 없는 TTS 모델 'VoxCPM2' 오픈소스 공개
**출처:** [AIToolly — 2026-04-11](https://aitoolly.com/ai-news/article/2026-04-11-openbmb-launches-voxcpm2-a-tokenizer-free-text-to-speech-model-for-multilingual-voice-generation-and)

OpenBMB가 20억 파라미터 규모의 토크나이저-프리 TTS 모델 'VoxCPM2'를 공개했다. AudioVAE V2의 잠재 공간에서 직접 작동하는 확산-자기회귀(Diffusion Autoregressive) 구조로, 200만 시간 이상의 다국어 음성 데이터로 학습되었다. 30개 언어를 지원하며, 5~10분의 음성 샘플만으로 화자 적응(fine-tuning)이 가능하다. 텍스트 설명만으로 새로운 음성을 디자인하는 Voice Design 기능도 포함한다.

> **학습 연결:** 기존 TTS는 텍스트를 토크나이저로 서브워드 단위로 분할한 뒤 Transformer로 처리하지만, VoxCPM2는 토크나이저를 제거하고 잠재 공간(latent space)에서 직접 음성을 생성한다. Ch04에서 배우는 토크나이저(BPE, WordPiece)의 역할과 한계를 이해한 뒤, 이를 우회하는 최신 접근법이 왜 등장했는지 비교해볼 수 있는 사례다.

### 2. 뉴엔AI 'QuettaLLMs', 과기부 K-AI 리더보드 종합 1위 등극
**출처:** [ZDNet Korea — 2026-04-10](https://zdnet.co.kr/view/?no=20260410111700)

한국 AI 스타트업 뉴엔AI의 'QuettaLLMs(27B-Koreasoner-V3)'가 과학기술정보통신부 K-AI 리더보드에서 종합 1위를 차지했다. 한국어 문화·언어 이해도를 측정하는 CLIcK에서 0.794점, 전문직 자격시험 기반 고난도 추론 KMMLU-Pro에서 0.676점을 기록했다. 270억 파라미터 규모로 대형 모델 대비 학습 데이터 품질 최적화 전략이 주효했다는 평가다.

> **학습 연결:** K-AI 리더보드의 CLIcK 벤치마크는 한국어 특유의 조사·어미 변화, 문화적 맥락을 모델이 얼마나 정확히 이해하는지 평가한다. Ch04에서 배우는 한국어 형태소 분석과 토크나이저 설계가 모델 성능에 직접적으로 영향을 미치며, 특히 교착어인 한국어에서 서브워드 토크나이징 전략(형태소 단위 vs BPE)의 선택이 벤치마크 점수를 좌우한다.

### 3. Anthropic, 'Advisor Strategy' 발표 — 대형 Transformer가 소형 모델을 실시간 코칭
**출처:** [Anthropic Blog — 2026-04-09](https://claude.com/blog/the-advisor-strategy)

Anthropic이 Messages API에 'Advisor Strategy'를 도입했다. Claude Opus 4.6이 어드바이저 역할로 전략적 가이던스를 제공하고, Sonnet 4.6 또는 Haiku 4.5가 실행자로 실제 태스크를 수행하는 이중 구조다. SWE-bench Multilingual에서 Sonnet 단독(72.1%) 대비 Opus 어드바이저 결합 시 74.8%(+2.7%p)를 달성하면서 태스크당 비용은 11.9% 절감했다. 어드바이저는 상담 당 400~700토큰만 생성한다.

> **학습 연결:** 이 전략은 두 Transformer 모델의 어텐션 레이어가 서로 다른 추상화 수준에서 동일 컨텍스트를 해석하는 구조다. 대형 모델의 깊은 셀프 어텐션이 문제의 전략적 구조를 파악하고, 소형 모델의 어텐션이 세부 실행에 집중한다. Ch04에서 배우는 멀티헤드 어텐션의 '서로 다른 헤드가 서로 다른 패턴에 주목'하는 원리가 모델 간 협업으로 확장된 사례다.

### 4. Perplexity, AI 에이전트 전환으로 ARR $4.5억 돌파 — 한 달 만에 50% 급증
**출처:** [PYMNTS — 2026-04-10](https://www.pymnts.com/artificial-intelligence-2/2026/perplexitys-shift-to-ai-agents-boosts-revenue-50/)

Perplexity의 연간 반복 매출(ARR)이 3월 기준 $4.5억을 넘어 한 달 만에 50% 급증했다. 월간 활성 사용자 1억 명, 기업 가치 약 $200억이다. 에이전트형 도구 'Computer' 출시와 사용량 기반 과금 모델 전환이 성장 동력이며, AI 검색에서 AI 에이전트 플랫폼으로의 전환이 매출 구조를 근본적으로 바꿨다.

> **학습 연결:** Perplexity의 검색-답변 파이프라인은 사용자 쿼리를 토크나이징→임베딩→검색 문서와의 코사인 유사도 계산→어텐션 기반 문맥 종합이라는 Ch04 NLP 파이프라인 전체를 실시간 서비스로 구현한 사례다. 특히 쿼리와 문서 간 의미적 매칭에 BERT 스타일 임베딩이 핵심 역할을 한다.

### 5. Shopify, AI Toolkit 출시 — AI 에이전트가 이커머스 스토어를 직접 운영
**출처:** [Shopify Developer Changelog — 2026-04-09](https://shopify.dev/changelog/shopify-ai-toolkit-connect-your-ai-tools-to-the-shopify-platform)

Shopify가 AI Toolkit을 오픈소스(MIT 라이선스)로 공개했다. Claude Code, OpenAI Codex, Cursor, Gemini CLI 등 AI 코딩 에이전트가 자연어로 상품 관리, 재고 업데이트, 스토어 운영을 직접 수행할 수 있다. 16개 스킬 파일과 실시간 API 스키마 검증, MCP(Model Context Protocol) 기반 도구 통합을 제공한다. 주요 이커머스 플랫폼이 외부 AI 에이전트에 실제 운영 권한을 부여한 최초 사례다.

> **학습 연결:** AI Toolkit에서 자연어 명령("재고를 50개로 업데이트해줘")이 API 호출로 변환되는 과정은 의도 파악(intent classification)→개체명 인식(NER)→명령어 생성이라는 NLP 태스크 체인이다. Ch04에서 배우는 시퀀스 라벨링과 Transformer 기반 텍스트 분류가 이런 실무 자동화의 기반 기술이 된다.
