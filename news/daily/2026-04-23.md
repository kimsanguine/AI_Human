### 1. Google, Gemini in Chrome을 한국·일본 포함 APAC 7개국으로 확장
**출처:** [TechCrunch — 2026-04-20](https://techcrunch.com/2026/04/20/google-rolls-out-gemini-in-chrome-in-seven-new-countries/)

구글이 Chrome 브라우저의 Gemini 통합을 호주·인도네시아·일본·필리핀·싱가포르·한국·베트남 등 APAC 7개국으로 확대했다. 주소창·사이드바에서 자연어로 질문하고, 페이지 요약·번역·비교 작업을 한국어로 직접 수행할 수 있다. 일본을 제외한 모든 국가는 데스크톱과 iOS에서 동시 제공된다.

> **학습 연결:** Ch04에서 배우는 다국어 토크나이저(BPE, SentencePiece)와 크로스링구얼 임베딩이 실제 브라우저 경험에서 어떻게 작동하는지 보여준다. 한국어처럼 형태소 경계가 모호한 언어는 subword tokenization의 설계가 응답 품질을 직접 좌우한다.

### 2. Anthropic, Claude Opus 4.7 공개 — Transformer 기반 agentic 성능 재탈환
**출처:** [VentureBeat — 2026-04-16](https://venturebeat.com/technology/anthropic-releases-claude-opus-4-7-narrowly-retaking-lead-for-most-powerful-generally-available-llm)

Anthropic이 Opus 4.7을 출시하며 agentic coding·scaled tool-use·agentic computer use 벤치마크에서 GPT-5.4와 Gemini 3.1 Pro를 근소한 차이로 재역전했다. "rigor" 개념을 도입해 모델 스스로 검증 단계를 설계한 뒤 작업 완료를 선언하도록 튜닝했고, 이미지 입력 해상도는 최대 2,576px(약 3.75MP)까지 3배 확장됐다. AWS Bedrock·Vertex AI·Microsoft Foundry에서 동시 제공된다.

> **학습 연결:** Ch04에서 배우는 self-attention 메커니즘이 멀티모달 입력(텍스트+고해상도 이미지)과 장문 컨텍스트로 확장될 때 position embedding과 토큰 병합 전략이 핵심이 된다. "rigor" 튜닝은 RLHF 이후 단계의 self-verification 기법이 실제 프로덕트에 어떻게 녹아드는지를 보여준다.

### 3. Google, Cloud Next에서 Gemini Enterprise Agent Platform 공개

### 4. Amazon, Anthropic에 추가 50억 달러 투자 — Anthropic은 AWS에 10년 1,000억 달러 약정
**출처:** [TechCrunch — 2026-04-20](https://techcrunch.com/2026/04/20/anthropic-takes-5b-from-amazon-and-pledges-100b-in-cloud-spending-in-return/)

Amazon이 Anthropic에 50억 달러를 추가 투자해 누적 투자액이 130억 달러에 달했다. 동시에 Anthropic은 향후 10년간 AWS에 1,000억 달러 이상을 집행하고, 최대 5GW의 신규 컴퓨팅 자원을 확보하기로 했다. Anthropic의 연환산 매출은 2025년 말 90억 달러에서 2026년 3월 말 300억 달러로 폭증했다.

> **학습 연결:** 프런티어 모델 학습·추론 원가는 GPU 공급 계약 구조가 결정한다. CPO·PM 관점에서 모델 가격 책정, 원가 구조, 장기 capacity 확보는 제품 전략의 핵심 변수가 된다.
