## TODAY'S TOP AI NEWS

### 1. OpenAI, 보안 특화 AI 에이전트 'Codex Security' 리서치 프리뷰 공개
**출처:** [OpenAI Blog](https://openai.com/index/codex-security-now-in-research-preview/)

OpenAI가 3월 6일 코드 보안 전문 AI 에이전트 Codex Security를 리서치 프리뷰로 공개했다. 이 에이전트는 레포지토리 전체를 분석해 프로젝트별 위협 모델을 생성하고, 취약점을 탐지·검증·패치까지 수행한다. 베타 기간 동안 120만 건의 커밋을 스캔해 792건의 Critical, 10,561건의 High-severity 이슈를 발견했으며, 오탐률을 50% 이상 줄였다.

> **학습 연결:** Python 코드를 작성할 때부터 보안을 고려하는 습관이 중요하다. Codex Security 같은 도구가 코드의 보안 취약점을 자동으로 찾아주지만, 기본적인 입력 검증이나 에러 처리 패턴은 Ch01에서부터 직접 익혀야 한다.

### 2. Apple, AI 기반 차세대 Siri 3월 정식 출시 — 컨텍스트 인식 어시스턴트로 진화
**출처:** [Crescendo AI](https://www.crescendo.ai/news/latest-ai-news-and-updates)

Apple이 완전히 재설계된 AI 기반 Siri를 3월 중 정식 출시한다고 발표했다. 화면 인식(on-screen awareness)과 앱 간 원활한 연동이 핵심으로, 단순 음성 명령을 넘어 사용자의 현재 컨텍스트를 이해하는 어시스턴트로 탈바꿈한다. AI 어시스턴트 시장에서 Google, OpenAI와의 경쟁이 본격화되는 신호탄이다.

> **학습 연결:** Siri 같은 AI 어시스턴트의 백엔드도 Python으로 프로토타이핑되는 경우가 많다. 자연어 처리, API 연동, 데이터 파이프라인 등 뒤에서 다룰 주제들의 실제 적용 사례를 미리 눈여겨보자.

### 4. Python 3.15 Alpha 7 릴리스 — frozendict 내장 타입 도입
**출처:** [Real Python](https://realpython.com/python-news-march-2026/)

Python 3.15.0 alpha 7이 오늘(3/10) 릴리스 예정이다. 가장 주목할 변화는 frozendict(불변 딕셔너리)가 내장 타입으로 추가되는 것과, 컴프리헨션 언패킹 문법의 도입이다. 5월 5일 베타 프리즈 전까지 추가 기능이 포함될 수 있으며, 타입 체커 ty 0.7.2도 대규모 AI 프로젝트의 빠른 타입 검사를 지원한다.

> **학습 연결:** 지금 배우는 딕셔너리(dict)의 확장 개념이 frozendict다. Ch01에서 자료구조를 배울 때 "변경 가능(mutable) vs 불변(immutable)"의 차이를 확실히 이해해두면, 이런 새로운 기능도 자연스럽게 받아들일 수 있다.

### 5. Anthropic 연구 — AI 코딩 어시스턴트가 개발자 스킬 형성에 미치는 영향
**출처:** [Anthropic Research](https://www.anthropic.com/research/AI-assistance-coding-skills)
