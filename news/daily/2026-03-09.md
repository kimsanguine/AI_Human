### 4. Astral의 uv 패키지 매니저, 월 7500만 다운로드 돌파 — pip 대체 가속
**출처:** [Astral Blog](https://astral.sh/blog)

Rust로 작성된 Python 패키지 매니저 uv가 PyPI 월간 다운로드 7500만 건을 돌파하며 Poetry(약 6600만)를 넘어섰다. pip 대비 10~100배 빠른 설치 속도를 자랑하며, 같은 팀이 만든 Ruff(린터)와 ty(타입체커)까지 합쳐 Astral이 Python 도구 생태계의 새로운 표준을 세우고 있다. 2026년 신규 프로젝트에서는 uv가 기본 권장 도구로 자리잡았다.

> **학습 연결:** Ch01에서 Python 환경을 설정할 때 pip 대신 uv를 사용해보자. `uv pip install` 한 번이면 체감 속도 차이를 바로 느낄 수 있고, 실무에서 바로 통하는 현대적 워크플로우를 처음부터 익힐 수 있다.

### 5. VS Code 1.110 업데이트 — AI Agent 확장성 및 브라우저 자동화 도구 추가
**출처:** [Microsoft DevBlogs](https://devblogs.microsoft.com/python/python-in-visual-studio-code-february-2026-release/)

VS Code 2026년 2월 릴리스(1.110)에서 AI 에이전트 확장성(Agent extensibility), 브라우저 구동 채팅 도구, 확장된 채팅 접근성 기능이 추가되었다. GitHub Copilot과의 통합이 더욱 깊어지면서, 코드 생성부터 테스트, 디버깅까지 AI가 개발 워크플로우 전반에 관여하는 시대가 열리고 있다.

> **학습 연결:** 지금 VS Code를 설치하고 Python 확장팩을 설정하는 것이 Ch01의 핵심 과제다. GitHub Copilot Free 티어를 활성화하면 코드 작성 시 AI의 도움을 바로 체험할 수 있다.
